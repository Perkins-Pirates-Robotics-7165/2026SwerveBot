# Controls

Here is the list for what each button does on the controller  
It would be nice to have a graphic, but I don't really know how to do that (sorry)

## Primary Controller

### P Triggers

| Button | Action | Run Type | Command |
| --------------- | --------------- | --------------- | --------------- |
| Left Trigger | Reverse Shoot | While True | Shoot |
| Right Trigger | Shoot | While True | Shoot |

### P Bumpers

| Button | Action | Run Type | Command |
| --------------- | --------------- | --------------- | --------------- |
| Left Bumper | Brake | While True | SwerveRequest.SwerveDriveBrake |
| Right Bumper | **```Unassigned```** | **```Unassigned```** | **```Unassigned```** |

### P Sticks (Free)

| Button | Action | Run Type | Command |
| --------------- | --------------- | --------------- | --------------- |
| Left Stick | Drive | Default Command | SwerveRequest.FeildCentric |
| Right Stick | Turn | Default Command | SwerveRequest.FeildCentric |

### P Sticks (Pressed)

| Button | Action | Run Type | Command |
| --------------- | --------------- | --------------- | --------------- |
| Left Stick | **```Unassigned```** | **```Unassigned```** | **```Unassigned```** |
| Right Stick | **```Unassigned```** | **```Unassigned```** | **```Unassigned```** |

### P Buttons

| Button | Action | Run Type | Command |
| --------------- | --------------- | --------------- | --------------- |
| A | Intake | While True | Intake |
| B | Bump | While True | Bump |
| X | Reverse Bump | While True | Bump |

### P POV Buttons
