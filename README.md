# 2026SwerveBot

Our Swerve drive robot for the 2026 frc season, "Rebuilt."

## Perkins Pirate Robotics

## Features

The current features of the our robot:
- Over-Bumper Ball intake
  - Can be raised/lowered by two side arms
- Central Ball Storage Bin
- 3-Stage Ball Shooter

### Possible Features in the Future

- Multi-Stage Climber
- Limelight Aiming-Assist for Shooting

## Components

- 4 Wheel Swerve MK4i Drivetrain
  - Kraken Drive Motors
  - Kraken Swerve Motors
  - CanCoders Encoders
- 3-Stage Ball Shooter
  - Bin Syphon to bring the balls into the shooter
  - Fins to bump the balls upward
  - Wheels to catch the balls
- 30-Wheel Intake
  - Throws the balls over the bumper into the bin
  - Can Intake 3-4 Balls at Once
